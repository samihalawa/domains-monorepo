## 🔧 Technical Setup

### Current Infrastructure
- **Main Platform**: Cloudflare Pages
- **Router**: `domains-router.js` 
- **Deploy**: `deploy.sh` script

### Project Structure
```

├── sites/                    # All domain sites
│   ├── [domain]/            # Individual site folders
│   └── default/             # Fallback site
```
